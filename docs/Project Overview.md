
# GovWatch Bangalore

## What we're building
A dashboard for government officials showing civic issues 
posted on Twitter/X for Bangalore.

## Tech Stack
- Scraping: Manual seed → Apify later
- AI: Claude API (classification in Kannada/Hindi/English)
- Backend: Python + FastAPI
- Database: PostgreSQL
- Frontend: React + Tailwind
