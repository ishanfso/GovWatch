# GovWatch — Bangalore Civic Intelligence Dashboard

A real-time dashboard that surfaces civic complaints posted on Twitter/X by Bangalore citizens, giving government officials a clear view of what needs attention.

---

## What It Does

1. **Collects** public tweets about civic issues in Bangalore (potholes, flooding, power cuts, waste, etc.)
2. **Categorizes** them automatically by issue type and area
3. **Displays** them in a clean dashboard built for government officials

## Who It's For

Government officials and civic administrators in Bangalore who need to quickly understand what issues citizens are raising online.

## Current Status

MVP in development. See [`docs/BUILD_LOG.md`](docs/BUILD_LOG.md) for what's live.

## Quick Start (For Developers)

1. Clone the repo
2. Open `dashboard/index.html` in a browser — works immediately with sample data
3. For live Twitter data, see [`docs/SETUP_GUIDE.md`](docs/SETUP_GUIDE.md)
